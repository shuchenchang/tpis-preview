# TPIS Preview

**Taipei Policy Intelligence Search**  
**臺北市政策智慧分析平台**

TPIS Preview is a v0.1 public showcase for a policy intelligence interface built around Taipei City public policy texts.

This repository is only a static product showcase. It demonstrates the product concept, data foundation, and module outcomes without publishing core analysis code, model instructions, credentials, or live search functionality.

## What This Preview Includes

- A Streamlit showcase landing page
- Static previews for Modules 1-5
- Public CSV data used for summary display
- A lightweight data loader for reading dataset summary statistics

## What This Preview Does Not Include

- Core analysis logic
- Model instructions
- Credentials
- Live policy search
- Live AI question answering
- Colab source notebooks or raw modules

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
